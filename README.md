## What is IAN?
IAN is the symbol for Ian Coin, a Solana-based token fueled by memes and dreams (and discord shenanigans).
One Ian is worth exactly 1 Ian.

## Planned features
- Custodial model for tracking transactions. All balances and transactions are managed via a PostGres server housed at an undisclosed location (probably Lee's computer and only when he feels like it).
- Withdrawals, deposits, investment, purchases, account creation, and more all facilitated by a Discord.Net bot and .NET backend
- A golden retriever-based LLM for managing large-scale economic forces. Inflation events, monetary freezes, and other terms I don't understand will be managed entirely by AI with no hope for human intervention.
- Purchase useless things with Ian coin! Send prompts to the overlord LLM to beg for mercy, purchase dog-related NFTs, MYSTERY BOX
